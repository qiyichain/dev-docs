## 是什么？

Rarible Multichain Protocol 是一个去中心化的工具集，可简化开发人员使用 NFT 的方式。协议为多个区块链构建一个抽象层，并使用 Multichain SDK 将开发人员与他们的细节隔离开来。

Rarible 多链 SDK 使应用程序能够轻松地与 Rarible 协议交互：在任何支持的区块链上查询、发行和交易 NFT。
