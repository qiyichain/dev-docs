# 票务系统


票务系统设计: https://www.woshipm.com/pd/3651828.html

## 几个概念

- event: 事件, 某时发生在某地的某件事
- organizer: 组织, 抽象出来的运行事件的实体
- user: 用户，所有能够登录管理后台管理事件的人
- items: 产品， 即门票
- variations: 每个items可以不同款式
- quotas: 定额，item可以被卖出的次数
- voucher： 代金券， 可以有优惠
