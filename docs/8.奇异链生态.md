# 奇异链生态

- QNS(Qiyichain Name Service): 对标ENS的域名服务
- NFT: 支撑稀物、稀台、及国内其他NFT项目方
- BaaS: 提供区块链服务平台


## QNS (Qiyichain Name Service)

QNS 是奇异链域名服务，主要实现：

- 去中心化方式的实现（基于智能合约）
- 中心化交易（在稀物）
- 转移、绑定、解析
- SDK支持，可以在JavaSDK中支持QNS的解析



> 思考：在稀物稀台中支持QNS方式的转移，目的地址是否可以是在DID中不存在，即目的地址可能是外部平台的用户，只是用了奇异链和QNS，并没有在DID中注册。
